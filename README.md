# 참! 잘했어요
함께 봉사할 인원을 찾고 정보를 공유하는 비영리 안드로이드앱 '참! 잘했어요'의 백엔드서버입니다.

![images](https://github.com/mohyeonMan/GreatJob/assets/112916714/72ddc658-f874-493b-bb3f-30480edf6867)


## 구성원
- 안드로이드앱 - 박준영
- 서버 - 박지훈

## 개발 환경
- Java 8
- SpringBoot 2.7.6
- Maven
- MyBatis
- MySQL
- STS 4, MySQL Workbench
- heroku, AWS S3
- Git, Github
