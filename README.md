# 참! 잘했어요

## 구성원
- 안드로이드앱 - 박준영
- 서버 - 박지훈

## 개발 환경
- Java 8
- SpringBoot 2.7.6
- Maven
- MyBatis
- MySQL
- STS 4, MySQL Workbench
- heroku, AWS S3
- Git, Github
